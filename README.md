# Linux Security Agent & Dashboard

A lightweight endpoint compliance monitoring solution that continuously audits Linux systems against CIS benchmark controls.

## Stack
- Go Agent
- Node.js + Express API
- SQLite
- React + Vite Dashboard

## Features
- Host inventory collection
- Installed package discovery
- 10 CIS Level-1 security checks
- Pass/Fail evidence reporting
- Real-time compliance dashboard
- Local-first architecture (AWS-ready future upgrade)
